## ✨ Overview

**Royal Cravings** is a sleek, responsive, and interactive **food ordering web interface** designed to deliver a premium user experience.

The project replicates real-world food delivery platforms with a focus on **clean UI, smooth interactions, and dynamic cart functionality** — making it a strong frontend showcase project.

---

## 🎯 Key Highlights

* 🛒 **Dynamic Cart System** – Real-time item updates
* 📱 **Fully Responsive Design** – Works across all devices
* 🎨 **Modern UI/UX** – Clean layout with smooth animations
* ⚡ **Fast & Lightweight** – Pure vanilla implementation
* 🧩 **Modular Code Structure** – Easy to scale and maintain

---

## 🧠 What I Built & Learned

This project demonstrates:

* Building **real-world UI components** (cart, menu, sections)
* Managing **state using JavaScript (cart logic)**
* Creating **responsive layouts using CSS Flexbox/Grid**
* Designing **user-focused experiences**
* Structuring a project for **scalability and readability**
